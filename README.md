# Building Risk Profiling

A prototype risk profiling tool for UK residential blocks, focused on high-rise and mid-rise housing.

## Purpose
This MVP demonstrates how a portfolio of buildings can be ranked by risk using a simple, explainable scoring model.

## Core risk themes
- Fire and building safety
- Damp and mould
- Compliance
- Repairs and maintenance
- Complaints and resident signal
- Financial exposure

## MVP outputs
- Total risk score
- Portfolio rank
- Risk drivers
- Recommended actions
